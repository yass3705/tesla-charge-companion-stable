# Mise à jour manuelle des tarifs Tesla

1. Ouvre Tesla Charge Companion.
2. Va dans l’onglet **Bornes**.
3. Repère le Superchargeur concerné.
4. Clique sur **Fiche Tesla** pour consulter le tarif officiel.
5. Reviens dans l’application et clique sur **Modifier**.
6. Dans **Tarification** :
   - choisis toute la journée ou un créneau horaire ;
   - sélectionne tarif au kWh ou à la minute ;
   - saisis le prix ;
   - vérifie la devise ;
   - ajoute les frais éventuels ;
   - ajoute d’autres créneaux avec **Ajouter un tarif** si nécessaire.
7. Mets à jour la date **Dernière mise à jour**.
8. Clique sur **Enregistrer**.

La modification est enregistrée dans le navigateur de l’appareil. Elle n’est pas automatiquement propagée aux autres appareils.
