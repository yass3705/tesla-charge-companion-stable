# Test de la synchronisation GitHub

1. Créer un jeton GitHub à accès fin limité au dépôt du Companion.
2. Donner uniquement la permission **Contents: Read and write**.
3. Dans le Companion, ouvrir **Synchronisation**.
4. Renseigner le propriétaire, le dépôt, la branche et le chemin `data/custom_stations.json`.
5. Coller le jeton, enregistrer puis toucher **Synchroniser maintenant**.
6. Répéter la configuration sur chaque appareil.

Le jeton ne doit jamais être ajouté au dépôt. Il reste dans le stockage local du navigateur.
