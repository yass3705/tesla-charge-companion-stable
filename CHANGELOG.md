# Changelog

## V6.6 stable

- Nouveau mode : tarification à la minute selon la puissance délivrée.
- Nombre illimité de tranches de puissance configurables.
- Calcul minute par minute à partir de la courbe de charge estimée.
- Frais fixes/parking dans la devise du tarif.
- Frais de congestion ou d’occupation après la charge.
- Ajout de l’exemple Tesla Casablanca — Onomo :
  - 0–60 kW : 1,10 MAD/min ;
  - 60–100 kW : 2,30 MAD/min ;
  - 100–150 kW : 3,50 MAD/min ;
  - parking : 6 MAD ;
  - congestion : jusqu’à 4,50 MAD/min.
- Géocodage international et saisie directe de coordonnées GPS.
- Dépôt cible : `tesla-charge-companion-stable`.
